3) Fazer as conversões de decimal para a base indicada:

a) 54(10) =(4)
b) 67(10) =(8)
c) 76(10) =(16)
d) 157(10) =(16)
e) 731(10) =(16)

a)  54 / 4 = 13 + 2
    13 / 4 = 3 + 1
    3 / 4  = 0 + 3
    = 312(4)

b)  67(10) / 8 = 8 + 3

    1  / 8 = 0 + 1
    =103(8)

c)  76 / 16 = 4  + 12

    = 4C(16)

d)  157 / 16 = 9 + 13
    9   / 16 = 0 + 9
    = 9D(16)

e)  731 / 16 = 45 + 11
    45  / 16 = 2  + 13
    2   / 16 = 0  + 2
    = 2DB(16)

    = 16 + 8 + 0 + 2 + 1 =  27 / 8 = 3 + 3
                            3  / 8 = 0 + 3
                            = 33(8)
c) 100100(2)
1x2^5 0x2^4 0x2^3 1x2^2 0x2^1 0x2^0
    = 32 + 0 + 0 + 4 + 0 + 0 = 36 / 16 = 2 + 4
                                2 / 16 = 0 + 2
                                = 24(16)

    = 32 + 0 + 8 + 0 + 2 + 1 = 43 / 8 = 5 + 3
                                5 / 8 = 0 + 5 
                                = 53(8)
